# Instant Teleport

Instant Teleport is a Dalamud plugin for FFXIV that makes Teleport and Return feel faster.

## What it does

- Removes Teleport/Return cast time.
- Reduces transition/fade delay between zones.
- Optionally suppresses transition UI elements.

## Command

- `/instanttp` opens plugin settings.
- `/instanttp status` prints current feature state.
- `/instanttp debug` toggles debug logging.

## Install

Add this custom repo URL in Dalamud:

`https://raw.githubusercontent.com/frenchpresssscoffee-blip/skip/442aeab/repo-v2.json`

## Notes

Use at your own risk and only in environments you are authorized to modify.
